API Reference

Core API

h(type, props, children)

Creates a virtual node (VNode) representing a DOM element.

Parameters:

  • type (string) - HTML element tag name
  • props (object) - Element properties and attributes
  • children (array|string) - Child vnodes or text content

Returns: VNode object

const vnode = h('div', { class: 'container' }, [
  h('h1', {}, 'Hello World'),
  h('button', { onclick: handleClick }, 'Click Me')
]);

render(vnode, container)

Renders a virtual node tree to a real DOM container with automatic diffing.

Parameters:

  • vnode (VNode) - Virtual node to render
  • container (HTMLElement) - DOM element to render into
render(vnode, document.getElementById('app'));

State Management

createStore(reducer, initialState, enhancer)

Creates a Redux-style state store with middleware support.

Parameters:

  • reducer (function) - Reducer function
  • initialState (object) - Initial state object
  • enhancer (function) - Optional store enhancer

Returns: Store object with methods:

  • getState() - Get current state
  • dispatch(action) - Dispatch an action
  • subscribe(listener) - Subscribe to state changes
const store = createStore(reducer, { todos: [] });
store.subscribe(() => render(App(store.getState()), root));
store.dispatch({ type: 'ADD_TODO', payload: todo });

Routing

Router(routes)

Sets up hash-based routing with parameter extraction.

Parameters:

  • routes (object) - Route patterns mapped to handler functions

Methods:

  • navigate(path) - Navigate to a specific route
  • getCurrentRoute() - Get current route information
const router = new Router({
  '/': homeHandler,
  '/todos/:filter': filterHandler
});
router.navigate('/todos/active');

Event System

Event Delegation

The framework uses pure global event delegation. All events are handled through event properties on vnodes.

Supported Events:

  • onclick - Click events
  • onchange - Change events
  • oninput - Input events
  • onkeydown - Keyboard events
  • ondblclick - Double-click events
  • And more...
h('button', {
  onclick: (e) => console.log('Clicked!'),
  onmouseenter: (e) => console.log('Mouse entered')
}, 'Click Me')
