Getting Started

MiniFramework is a lightweight JavaScript framework built from scratch to demonstrate modern web development concepts. It includes:

  • Virtual DOM with diffing algorithm
  • Component-based architecture
  • State management system
  • Client-side routing
  • Event handling system

API Reference

framework.createElement(tag, props, ...children)

Creates a virtual DOM element.

createElement('div', { className: 'container' }, 'Hello World')

framework.render(element, container)

Renders a virtual DOM element to a real DOM container.

framework.render(element, document.getElementById('root'))

framework.createStore(initialState, reducers)

Creates a state management store with reducers.

const store = framework.createStore({ count: 0 }, { INCREMENT: (state) => ({ ...state, count: state.count + 1 }) })

framework.route(path, handler)

Sets up a route with a handler function.

framework.route('/', () => console.log('Home page'))
