Getting Started

MiniFramework is a lightweight JavaScript framework built from scratch to demonstrate modern web development concepts. It includes:

  • Virtual DOM with O(n) diffing algorithm
  • Key-based list reconciliation
  • Component-based architecture with lifecycle hooks
  • Redux-style state management
  • Client-side routing
  • Global event delegation system

Quick API Reference

h(type, props, children)

Creates a virtual node (VNode) representing a DOM element.

const vnode = h('div', { class: 'container' }, [
  h('h1', {}, 'Hello World'),
  h('button', { onclick: handleClick }, 'Click Me')
]);

render(vnode, container)

Renders a virtual node tree to a real DOM container with automatic diffing.

render(vnode, document.getElementById('app'));

createStore(reducer, initialState, enhancer)

Creates a Redux-style state store with middleware support.

const store = createStore(reducer, { todos: [] });
store.subscribe(() => render(App(store.getState()), root));
store.dispatch({ type: 'ADD_TODO', payload: todo });

Router(routes)

Sets up hash-based routing with parameter extraction.

const router = new Router({
  '/': homeHandler,
  '/todos/:filter': filterHandler
});
router.navigate('/todos/active');
